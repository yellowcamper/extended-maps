Detailed Description

template<class A, class B>
class extended::map< A, B >

The memory saving version of std::map<A, B>.

This class uses extended::null<A> to define a default_value for the std::map<A, B> unless it is changed in the constructor, then, it does not save items equal to the default_value and all values that are not defined will be defined as the default_value.

Note
All of the map<A, B> operators and functions are left unchanged, so there will not be any conflicts, however, many operations will use new operators to make use of this class.
A version of this class exists solely because std::string does not work like other objects, if you are using this code as a base for an application that acts like std::string, I would suggest making a version of the class to suit your needs better.
